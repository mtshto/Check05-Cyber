# Metodo_GET_POST

#### Verificar a Versão do PHP Instalado
```http
php -v
```

####  Instalar o PHP (se não estiver instalado)
```http
sudo apt update
sudo apt install php7.4 libapache2-mod-php7.4
```

#### Reiniciar o Apache
```http
sudo systemctl restart apache2
```
